# Submission Checklist

Use this file when you prepare screenshots, demo recordings, GitHub submission, and deployment links.

## Core Repo Submission

1. Keep the repository name as `react-essentials-assignment`.
2. Upload the full project to GitHub.
3. Add the live hosted link in the repository README after deployment.
4. Capture screenshots from the routes listed below.
5. Record a short walkthrough showing the main interactions.
